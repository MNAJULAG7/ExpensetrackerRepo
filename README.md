Expense Tracker
A simple full-stack Expense Tracker application to manage daily expenses efficiently.

Backend:
  Add new expense using DTO
  Get all expenses
  Delete expense by ID
Frontend:
  Form handling using React Hook Form
  Routing with React Router
  Styled using Tailwind CSS
  Clean and responsive UI

Tech Stack:
  Frontend
    React
    React Hook Form
    React Router
    Tailwind CSS
  Backend:  
    Java / Spring Boot
    REST APIs
    DTO pattern
    MySql DB
